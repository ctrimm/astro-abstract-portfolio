## Project Overview

FitLife wanted to create a mobile app that would help users track their fitness journey while connecting with a community of like-minded individuals.

## Key Features

- **Workout Tracking**: Log exercises, sets, reps, and weight
- **Progress Photos**: Visual timeline of transformation
- **Social Feed**: Share achievements and motivate others
- **Custom Plans**: AI-generated workout plans based on goals
- **Nutrition Tracking**: Calorie and macro counting

## Technical Implementation

Built with React Native for cross-platform compatibility, backed by a Node.js API and MongoDB database. Firebase handles real-time features like the social feed and notifications.

## Impact

The app has been downloaded over 50,000 times in the first three months with a 4.8-star rating on both iOS and Android platforms.
